A fun Android app for kids aged 3-6 to learn colors through interactive camera-based gameplay.

## Features

- **Color Detection**: Uses phone camera to detect colors in real-time
- **Mission System**: Random color challenges like "Find something red!"
- **Rewards**: Earn stars for successful color finds
- **Fun Sounds**: Playful sound effects when colors are found
- **Simple Animations**: Engaging visual feedback for kids

## Requirements

- Android API Level 16 (Android 4.1) or higher
- Camera hardware

## How to Play

1. Open the app
2. Read your mission (e.g., "Find something red!")
3. Point your camera at objects matching the color
4. When the color is detected, earn a star and hear a fun sound!
5. Complete missions to earn all 5 stars

## Technical Details

- Built with Kotlin
- Uses CameraX for camera functionality
- Real-time color analysis from camera feed
- Material Design components for UI

## Building

1. Open the project in Android Studio
2. Sync Gradle files
3. Build and run on a device or emulator with camera support
